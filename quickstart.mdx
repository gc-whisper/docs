# Quick Start Guide

Get the GameRamp Unity SDK up and running in your game in just a few minutes.

## Prerequisites

Before you begin, make sure you have:

<CardGroup cols={2}>
  <Card title="Unity 2021.3+" icon="unity">
    Unity 2021.3 LTS or higher installed
  </Card>
  <Card title="GameRamp Account" icon="user">
    SDK credentials from GameRamp
  </Card>
</CardGroup>

<Note>
  Don't have credentials yet? [Contact us](mailto:connect@gameramp.com) to get started.
</Note>

## Step 1: Import the SDK

<Steps>
  <Step title="Download Package">
    Download the GameRamp SDK (.unitypackage) from your dashboard
  </Step>
  <Step title="Import in Unity">
    In Unity: **Assets → Import Package → Custom Package**
  </Step>
  <Step title="Verify Installation">
    Confirm the `GamerampSDK/` folder appears in your project
  </Step>
</Steps>

## Step 2: Configure

<Steps>
  <Step title="Create Config">
    Right-click in Project window: **Create → Gameramp → SDKConfig**
  </Step>
  <Step title="Add Credentials">
    Enter your SDK Key and Secret Key in the Inspector
  </Step>
</Steps>

<Warning>
  Keep your Secret Key secure. Never commit it to public repositories.
</Warning>

## Step 3: Initialize

<Tabs>
  <Tab title="Using Prefab (Easiest)">
    1. Drag `Gameramp.prefab` from `GamerampSDK/Prefabs/` into your scene
    2. Assign your config to the prefab
    3. That's it! The SDK auto-initializes

    ```csharp
    // No code needed! The prefab handles everything
    ```
  </Tab>
  <Tab title="Manual Setup">
    ```csharp
    // 1. Initialize the SDK
    Gameramp.Initialize(config);

    // 2. Login
    Gameramp.Login(
        onSuccess: () => Debug.Log("Ready!"),
        onFail: () => Debug.LogError("Login failed")
    );
    ```
  </Tab>
</Tabs>

## Start Using GameRamp

Once initialized, you can start tracking:

<CodeGroup>
```csharp Analytics
// Track session start
Gameramp.TrackSessionStart();

// Track first open
Gameramp.TrackFirstOpen("organic", "direct");
```

```csharp Ads
// Track ad impressions
Gameramp.TrackBannerAd("menu_banner", "campaign_id");
Gameramp.TrackInterstitialAd("level_complete", "campaign_id");
Gameramp.TrackRewardedAd("extra_life", "campaign_id");
```

```csharp IAP
// Fetch products
Gameramp.FetchStoreProducts(
    (products) => {
        // Display products
    },
    (error) => {
        // Handle error
    }
);

// Make purchase
Gameramp.InitiatePurchase("product_id",
    onSuccess: (purchase) => {
        // Grant item
    },
    onFailed: (error) => {
        // Handle error
    }
);
```
</CodeGroup>

## Platform Setup

<Tabs>
  <Tab title="Android">
    - Minimum API Level 24 (Android 7.0)
    - Enable custom mainTemplate.gradle
    - Permissions auto-added
  </Tab>
  <Tab title="iOS">
    - Minimum iOS 12.0
    - StoreKit auto-included
    - Configure in Player Settings
  </Tab>
  <Tab title="IL2CPP">
    - Move `link.xml` to Assets/ root
    - Prevents code stripping
    - Required for builds
  </Tab>
</Tabs>
