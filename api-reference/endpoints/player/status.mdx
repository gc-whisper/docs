# Get Player Status

Retrieve AI-powered player predictions and status information. This endpoint provides insights about player behavior, such as whether they are likely to make purchases.

<Card title="Endpoint" icon="server">
  ```
  GET https://api.gameramp.com/v1/player/status
  ```
</Card>

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token obtained from login
  ```
  Authorization: Bearer <JWT_TOKEN>
  ```
</ParamField>

<ParamField header="X-Tenant-ID" type="string" required>
  Your tenant/studio identifier
</ParamField>

<ParamField header="X-Game-Id" type="string" required>
  Your game identifier
</ParamField>

<ParamField header="X-Player-Id" type="string" required>
  The player identifier to get status for
</ParamField>

```javascript Node.js
async function getPlayerStatus(token, tenantId, gameId, playerId) {
  const response = await fetch('https://api.gameramp.com/v1/player/status', {

    headers: {
      'Authorization': `Bearer ${token}`,
      'X-Tenant-ID': tenantId,
      'X-Game-Id': gameId,
      'X-Player-Id': playerId

  if (!response.ok) {
    throw new Error(`Failed to get player status: ${response.status}`);

    // Show standard offers

} catch (error) {
  console.error('Error:', error);
}
```

```csharp Unity/C#
using System.Collections;
using UnityEngine;
using UnityEngine.Networking;

[System.Serializable]
public class PlayerStatus

    [System.Serializable]
    public class PayerPrediction
    {
        public bool value;
        public bool defaultValue;
        public string expiredAt;
    }

    public PayerPrediction isPayer;
}

public class PlayerStatusManager : MonoBehaviour
{
    public IEnumerator GetPlayerStatus(
        string accessToken,
        string tenantId,
        string gameId,
        string playerId,
        System.Action<PlayerStatus> onSuccess,
        System.Action<string> onError)
    {
        string url = "https://api.gameramp.com/v1/player/status";

        using (UnityWebRequest request = UnityWebRequest.Get(url))
        {
            request.SetRequestHeader("Authorization", $"Bearer {accessToken}");
            request.SetRequestHeader("X-Tenant-ID", tenantId);
            request.SetRequestHeader("X-Game-Id", gameId);
            request.SetRequestHeader("X-Player-Id", playerId);

            yield return request.SendWebRequest();

            if (request.result == UnityWebRequest.Result.Success)
            {
                string json = request.downloadHandler.text;
                PlayerStatus status = JsonUtility.FromJson<PlayerStatus>(json);
                onSuccess?.Invoke(status);

                // Use prediction
                if (status.isPayer.value)
                {
                    Debug.Log("Player is likely to purchase");
                    ShowPremiumOffers();
                }
            }
            else
            {
                onError?.Invoke(request.error);
            }
        }
    }

    }
```

## Response

### Success Response (200 OK)

```json
{
  "isPayer": {
    "value": true,
    "defaultValue": false,
    "expiredAt": null
  }
}
```

### Response Fields

<ResponseField name="isPayer" type="object">
  Player payer prediction

  <Expandable title="isPayer properties">
    <ResponseField name="value" type="boolean" required>
      The current prediction value. `true` indicates the player is likely to make purchases
    </ResponseField>

    <ResponseField name="defaultValue" type="boolean" required>
      Default value to use if no prediction is available
    </ResponseField>

    <ResponseField name="expiredAt" type="string | null">
      ISO 8601 timestamp when this prediction expires, or `null` if no expiration
    </ResponseField>
  </Expandable>
</ResponseField>

## Caching Strategy

<Warning>
Player predictions should be cached appropriately to minimize API calls while ensuring data freshness.
</Warning>

### Recommended Caching

```javascript
class PlayerStatusCache {
  constructor() {
    this.cache = new Map();
    this.cacheTimeout = 3600000; // 1 hour in milliseconds
  }

  async getStatus(playerId, fetchFunction) {
    const cached = this.cache.get(playerId);

    // Check if cache is valid
    if (cached && Date.now() - cached.timestamp < this.cacheTimeout) {
      // Check if prediction hasn't expired
      if (!cached.data.isPayer.expiredAt ||
          new Date(cached.data.isPayer.expiredAt) > new Date()) {
        return cached.data;
      }
    }

    // Fetch fresh data
    const freshData = await fetchFunction(playerId);

    // Cache the result
    this.cache.set(playerId, {
      data: freshData,
      timestamp: Date.now()
    });

    return freshData;
  }

  invalidate(playerId) {
    this.cache.delete(playerId);
  }
}

// Usage
const statusCache = new PlayerStatusCache();

async function getPlayerStatusCached(playerId) {
  return statusCache.getStatus(playerId, async (id) => {
    // Fetch from API
    return await fetchPlayerStatusFromAPI(id);
  });
}
```
