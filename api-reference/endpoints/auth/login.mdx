# Login

Authenticate with GameRamp API using your API credentials and receive JWT tokens for subsequent requests. The login response also includes initial player configurations.

<Card title="Endpoint" icon="server">
  ```
  POST https://api.gameramp.com/auth/login
  ```
</Card>

## Request

### Headers

<ParamField header="Content-Type" type="string" required>
  Must be `application/json`
</ParamField>

### Body Parameters

<ParamField body="apiKey" type="string" required>
  Your API Key provided by GameRamp
</ParamField>

<ParamField body="deviceId" type="string" required>
  Unique device identifier (e.g., device UDID, installation ID, or generated UUID)
</ParamField>

<ParamField body="timestamp" type="string" required>
  Current UTC timestamp in ISO 8601 format: `YYYY-MM-DDThh:mm:ssZ`
</ParamField>

<ParamField body="signature" type="string" required>
  HMAC-SHA256 signature encoded in Base64. See [Authentication](/api-reference/authentication) for generation details.
</ParamField>

### Example Request

<CodeGroup>
```bash cURL
curl -X POST 'https://api.gameramp.com/auth/login' \
  -H 'Content-Type: application/json' \
  -d '{
    "apiKey": "test_tenant_4Dqms3ygG9FVC",
    "deviceId": "device-123",
    "timestamp": "2025-07-15T19:44:26Z",
    "signature": "/Sn9msNKK4ifTEYrGC/GKpRptolLIyWi4jsfdn5S4g8="
  }'
```

```javascript Node.js
const crypto = require('crypto');
const axios = require('axios');

async function login(apiKey, deviceId, apiSecret) {
  const timestamp = new Date().toISOString().replace(/\.\d{3}/, '');
  const payload = `${apiKey}|${deviceId}|${timestamp}`;

  const signature = crypto
    .createHmac('sha256', apiSecret)
    .update(payload)
    .digest('base64');

  const response = await axios.post('https://api.gameramp.com/auth/login', {
    apiKey,
    deviceId,
    timestamp,
    signature
  });

  return response.data;
}
```

```python Python
import hmac
import hashlib
import base64
import requests
from datetime import datetime

def login(api_key, device_id, api_secret):
    timestamp = datetime.utcnow().strftime('%Y-%m-%dT%H:%M:%SZ')
    payload = f"{api_key}|{device_id}|{timestamp}"

    signature = base64.b64encode(
        hmac.new(
            api_secret.encode('utf-8'),
            payload.encode('utf-8'),
            hashlib.sha256
        ).digest()
    ).decode('utf-8')

    response = requests.post(
        'https://api.gameramp.com/auth/login',
        json={
            'apiKey': api_key,
            'deviceId': device_id,
            'timestamp': timestamp,
            'signature': signature
        }
    )

    return response.json()
```
</CodeGroup>

### Response Fields

<ResponseField name="accessToken" type="string">
  JWT access token for authenticating API requests
</ResponseField>

<ResponseField name="refreshToken" type="string">
  Token for refreshing the access token when it expires
</ResponseField>

<ResponseField name="tokenType" type="string">
  Always "Bearer"
</ResponseField>

<ResponseField name="expiresIn" type="number">
  Access token validity period in seconds
</ResponseField>

<ResponseField name="userId" type="string">
  Unique user identifier
</ResponseField>

<ResponseField name="deviceId" type="string">
  The device ID used in the request
</ResponseField>

<ResponseField name="gameId" type="string">
  Game identifier
</ResponseField>

<ResponseField name="tenantId" type="string">
  Tenant/Studio identifier
</ResponseField>

<ResponseField name="playerConfig" type="object">

## Error Responses

### 401 Unauthorized

Invalid credentials or signature

```json
{
  "error": "Invalid authentication credentials",
  "status": "error"
}
```

### 400 Bad Request

Request timestamp outside acceptable window (Â±5 minutes)

```json
{
  "error": "Request timestamp is outside the acceptable time window",
  "status": "error"
}
```

## Security Notes

<Warning>
- API Secret must be kept secure and never exposed in client-side code
- Timestamps must be within 5 minutes of server time
- Each device should use a unique, persistent device ID
</Warning>
