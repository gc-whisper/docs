# Logout Device

Endpoint: **POST auth/logout**

Logs out a device by invalidating its active tokens. This endpoint requires authentication with a valid JWT token.

## Body

```json
{
  "deviceId": "string" // Required: The device identifier to logout
}
```

## Headers

- **Content-Type:** application/json
- **Authorization:** Bearer `<accessToken>`

## Response

### Success Response
```json
{
  "message": "Logout successful",
  "status": "success"
}
```

### Error Responses

- **401 Unauthorized** - Invalid or expired token
```json
{
  "error": "Invalid authentication credentials",
  "status": "error"
}
```

- **400 Bad Request** - Invalid device ID
```json
{
  "error": "Invalid device ID",
  "status": "error"
}
```

## Example

```bash
curl -X POST 'https://api.gameramp.com/auth/logout' \
  -H 'Authorization: Bearer eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...' \
  -H 'Content-Type: application/json' \
  -d '{
    "deviceId": "device-123"
  }'
```

Response:
```json
{
  "message": "Logout successful",
  "status": "success"
}
```
