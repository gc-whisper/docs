# Quick Start

Get started with GameRamp SDK in 3 steps:

## Step 1: Import the SDK

<Steps>
  <Step title="Download the Package">
    Download the GameRamp SDK package (.unitypackage).
  </Step>
  <Step title="Import in Unity">
    In Unity: Assets → Import Package → Custom Package.
  </Step>
  <Step title="Select and Import">
    Select and import the package.
  </Step>
  <Step title="Verify Installation">
    Verify `GamerampSDK/` folder is present.
  </Step>
</Steps>


## Step 2: Create Configuration

<Steps>
  <Step title="Create Config Asset">
    In the Project window: Create → Gameramp → SDKConfig.
  </Step>
  <Step title="Name the Config">
    Name it GamerampConfig.
  </Step>
  <Step title="Add Credentials">
    Fill in credentials in Inspector:
    - SDK Key (required)
    - Secret Key (required)
  </Step>
</Steps>


## Step 3: Initialize SDK

Choose one of the following initialization methods:

<Tabs>
  <Tab title="Option 1: Use the GameRamp Prefab (Recommended)">
    <Steps>
      <Step title="Add Prefab to Scene">
        Drag Gameramp.prefab from GamerampSDK/Prefabs/ into your first scene
      </Step>
      <Step title="Assign Configuration">
        Assign your GamerampConfig to the prefab's config field
      </Step>
      <Step title="Auto-initialization">
        The prefab will auto-initialize and auto-login on Start
      </Step>
    </Steps>
  </Tab>

  <Tab title="Option 2: Use GamerampSDK Component (Manual)">
    <Steps>
      <Step title="Create GameObject">
        Create an empty GameObject
      </Step>
      <Step title="Add Component">
        Add GamerampSDK.cs component
      </Step>
      <Step title="Assign Configuration">
        Assign your GamerampConfig
      </Step>
      <Step title="Initialize in Code">
        Call methods below before using any other features from SDK.

        ```csharp
        // 1. Initialize
        Gameramp.Initialize(config);
        // 2. Login
        Gameramp.Login(
            onSuccess: () => Debug.Log("Login successful!"),
            onFail: () => Debug.LogError("Login failed!")
        );
        ```
      </Step>
    </Steps>
  </Tab>
</Tabs>
