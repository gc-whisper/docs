# GameRamp Unity SDK

The GameRamp SDK is a lightweight Unity SDK that enables analytics, ads tracking, and in-app purchases (IAP) within your game.

## Supported Platforms

<CardGroup cols={3}>
  <Card title="Unity" icon="unity">
    Unity 2021.3+
  </Card>
  <Card title="Android" icon="android">
    Android 7.0+ (API Level 24)
  </Card>
  <Card title="iOS" icon="apple">
    iOS 12+
  </Card>
</CardGroup>

## Key Features

<CardGroup cols={2}>
  <Card title="Analytics" icon="chart-line" href="/sdks/unity/analytics">
    Track sessions, user behavior, and custom events to deliver a personalized experience
  </Card>
  <Card title="Personalized Ad Surfacing" icon="rectangle-ad" href="/sdks/unity/ads">
    Optimize interstitials and rewarded video surfacing so players have the best experience
  </Card>
  <Card title="In-App Purchases" icon="sack-dollar" href="/sdks/unity/iap">

## Why GameRamp SDK?

- **Lightweight**: Minimal impact on your game's performance
- **Easy Integration**: Simple 3-step setup process
- **Cross-Platform**: Works seamlessly on both Android and iOS
- **Comprehensive**: All essential monetization and analytics features in one SDK
- **Unity Native**: Built specifically for Unity developers

## Prerequisites

Before you begin, ensure you have:

<Steps>
  <Step title="Unity Version">
    Unity 2021.3 LTS or higher installed
  </Step>
  <Step title="Platform Requirements">
    - Android: Minimum SDK 24 (Android 7.0)
    - iOS: Minimum deployment target iOS 12.0
  </Step>
  <Step title="GameRamp Credentials">
    SDK Key and Secret Key from your GameRamp dashboard
  </Step>
</Steps>
