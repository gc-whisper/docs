# Introduction

The GameRamp SDK is a lightweight Unity SDK that enables analytics, ads tracking, and in-app purchases (IAP) within your game.

## Supported Platforms

- Unity 2021.3+
- Android 7.0+
- iOS 12+

## Key Features

- Analytics (sessions, first-open, custom events)
- Ads tracking (banner, interstitial, rewarded)
- In-App Purchases (IAP)
- Authentication & Player Config
- Store management
